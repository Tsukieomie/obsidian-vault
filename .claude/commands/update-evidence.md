description: Update Evidence Tracking with new evidence or verification status
---

Update the Evidence Tracking document with new evidence or change verification status.

Follow these steps:
1. Ask the user what they want to update:
   - Add new evidence item
   - Update verification status (verified, pending, investigating)
   - Add evidence to existing item
   - Update evidence quality assessment

2. Read Analysis/Evidence Tracking.md to see current state

3. For new evidence:
   - Get evidence description
   - Get source information
   - Get related entities
   - Assign initial verification status
   - Assess quality level (high/medium/low)

4. For status updates:
   - Find the evidence item
   - Update status
   - Add verification notes
   - Update quality assessment if needed

5. Update the document with proper formatting

6. Ask if they want to:
   - Update Evidence Repository.md with links
   - Update relevant entity profiles
   - Add to Timeline.md

After updating, provide a summary of changes made.
