---

Conduct a comprehensive weekly review of the investigation and update all relevant documents.

Follow these steps:

1. **Review Action Items**
   - Check completed items from past week
   - Identify incomplete high-priority items
   - Assess blockers or delays

2. **Review Evidence Tracking**
   - Check what moved from pending to verified
   - Identify stalled verification efforts
   - Note any evidence gaps discovered

3. **Review Timeline**
   - Summarize discoveries from past week
   - Identify patterns or trends
   - Note any significant events

4. **Review Entity Profiles**
   - Check for entities needing updates
   - Verify connection confidence levels
   - Update status fields as needed

5. **Update Analysis Documents**
   - Connection Map: Any new relationships?
   - Threat Assessment: Any threat level changes?
   - Risk Matrix: Any new risks or mitigations?
   - Intelligence Assessment: Any new indicators?

6. **Plan Next Week**
   - Review Action Items for next week priorities
   - Identify critical tasks
   - Note any deadlines

Generate a weekly review report including:
- Summary of past week's activities
- Key discoveries and updates
- Completed vs pending action items
- Evidence verification progress
- New entities or connections added
- Threat level changes
- Priorities for next week
- Recommendations

Save the review report with date in filename.
