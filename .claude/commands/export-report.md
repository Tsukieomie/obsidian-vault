description: Export a comprehensive report for external use (legal, law enforcement)
---

Generate a comprehensive investigation report for external use (legal, law enforcement, or documentation purposes).

Ask the user what type of report they need:
1. **Executive Summary** - High-level overview for decision makers
2. **Legal Report** - Court-ready documentation with evidence
3. **Law Enforcement Report** - For FBI/police with actionable intelligence
4. **Technical Report** - Deep technical analysis of infrastructure
5. **Entity Report** - Comprehensive profile of specific entity
6. **Full Investigation Report** - Complete documentation of everything

For each report type, compile relevant information:

**Executive Summary**:
- Current threat level and status
- Primary targets
- Key findings
- Intelligence assessment
- Recommendations

**Legal Report**:
- Executive summary
- Evidence repository with quality ratings
- Entity profiles with connection confidence
- Timeline of events
- Technical infrastructure documentation
- Sources and verification status

**Law Enforcement Report**:
- Threat assessment
- Intelligence assessment (80% probability)
- Entity profiles with criminal/intelligence indicators
- Technical infrastructure (IOCs, network analysis)
- Evidence compilation
- Recommended actions
- Urgency assessment

**Technical Report**:
- Complete network analysis (AS53616)
- Infrastructure mapping
- IOCs and indicators
- Technical capabilities
- Obfuscation techniques
- Monitoring recommendations

**Entity Report**:
- Selected entity full profile
- All connections
- All evidence
- Timeline of involvement
- Risk assessment
- Intelligence indicators

**Full Investigation Report**:
- All of the above
- Complete timeline
- All entities
- All evidence
- All analysis
- Complete recommendations

Format the report with:
- Professional structure
- Clear sections
- Evidence citations
- Confidence levels
- Executive summary at top
- Appendices for detailed data

Ask if they want the report:
- Saved as markdown in the vault
- With specific focus areas
- With sensitive information redacted
- For specific audience (judge, FBI agent, attorney, etc.)
