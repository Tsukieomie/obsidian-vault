description: Start a new investigation session with proper documentation
---

Start a new investigation session and create documentation for it.

Follow these steps:
1. Ask the user what they're investigating today
2. Create a new investigation note using Templates/Investigation Note Template.md
3. Set up the session with current date and investigator name
4. Guide the user through the investigation
5. Document findings as they work
6. Update relevant documents at the end

During the session:
- Document all findings with evidence and confidence levels
- Track entities involved
- Note questions that arise
- List next steps and verification needs

At the end of the session:
- Save the investigation note in Analysis/ folder
- Update Timeline.md with key discoveries
- Update Evidence Tracking.md with new evidence
- Update relevant entity profiles
- Update Action Items.md with new tasks
- Update Investigation Dashboard.md if significant findings

Provide a session summary showing:
- What was investigated
- Key findings
- Entities involved
- Evidence collected
- Next steps created
