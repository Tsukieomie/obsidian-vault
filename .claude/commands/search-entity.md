description: Search for an entity and show all related information
---

Search for an entity (person, organization, or infrastructure) and show comprehensive information about it.

Follow these steps:
1. Ask the user what entity they want to search for
2. Search across all vault files for mentions of the entity
3. Compile a comprehensive report including:
   - Entity profile (if exists)
   - All mentions in analysis documents
   - Connections to other entities
   - Evidence items related to this entity
   - Timeline entries involving this entity
   - Action items related to this entity
   - Risk assessment (if in Risk Matrix)

Present the information organized by:
- Profile Summary (from entity file)
- Connections (from Connection Map)
- Evidence (from Evidence Tracking and Evidence Repository)
- Timeline (chronological appearances)
- Risk & Threat (from Threat Assessment and Risk Matrix)
- Pending Actions (from Action Items)
- All Other Mentions (from other documents)

After presenting the information, ask if the user wants to:
- Update the entity profile
- Add new connections
- Update evidence status
- Create new action items
