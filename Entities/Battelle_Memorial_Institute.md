## DARPA Contracts

### N3 Program - BrainSTORMS
**Full Name:** Brain System to Transmit Or Receive Magnetoelectric Signals
**Award:** Part of $65 million N3 program (2018)
**Duration:** 4 years (2018-2022, extended)
**Goal:** Non-invasive, high-bandwidth brain-computer interface

### BrainSTORMS Technology

**Method:** Magneto-electric nanoparticles
**Delivery:** Injectable
**Function:** Bidirectional neural interface
**Control:** External magnetic field

**How it Works:**
```
1. Inject magnetic nanoparticles into brain
2. External magnetic field activates particles
3. Particles stimulate specific neurons
4. Deep brain regions accessible
5. Non-invasive control achieved
```

**CRITICAL:** This is remote brain control via injectable technology
